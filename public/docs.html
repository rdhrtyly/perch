Connector & API

Deploy to your Perch from Claude or your own scripts.

1. Make a deploy token

On the dashboard, open Deploy tokens → Create token. Copy it once — it's your key, treat it like a password.

2. Connect in Claude Code (CLI)

Add Perch as an MCP server with your token:

claude mcp add --transport http perch \
  https://useperch.dev/mcp \
  --header "Authorization: Bearer perch_YOUR_TOKEN"

3. Connect in Claude Desktop / Web

Settings → Connectors → Add custom connector → URL:

https://useperch.dev/connect

No token needed — it opens a Perch login page (log in with your email + password to authorize).

deploy_sitedeploy files (HTML or React/Next.js)

redeploy_siterebuild a site
check_siteconfirm a site is live + see its stats

delete_sitetear a site down (owner only)
